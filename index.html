Cartlens - Revolutionizing Retail with Computer Vision

Description

Cartlens revolutionizes retail by using computer vision to monitor store shelves, analyze basket contents, and understand customer behavior. It ensures real-time inventory management, detects popular co-purchased items, and tracks customer movement and dwell time. Leveraging technologies like Python, OpenCV, Roboflow, OpenVINO, and Awiros SDK, Cartlens provides scalable insights and visual analytics for efficient retail operations.

    • Objective: Enhancing retail processes by leveraging computer vision to automate and optimize key aspects of store management. Cartlens aims to streamline operations through real-time monitoring of product availability, in-depth analysis of shopping behaviors, and actionable insights to improve customer experience and inventory management. By utilizing advanced technologies, Cartlens seeks to provide a comprehensive solution that minimizes manual effort, reduces stockouts, and enhances the overall efficiency of retail operations.

    Use Cases:

    Shelf Monitoring:

    • Continuously monitor product levels on store shelves.
    • Alert staff when items are running low or out of stock, ensuring timely restocking and minimizing lost sales opportunities.

    Basket Analysis

    • Detect and analyze the items in customers' baskets.
    • Identify frequently co-purchased items using the Apriori algorithm to uncover shopping patterns and product associations.
    • Use these insights for targeted promotions and improved product placement.

    Dwell-Time Analysis:

    • Track customer movement throughout the store using object tracking.
    • Analyze the time customers spend in different sections to understand foot traffic patterns and areas of interest.
    • Use this data to optimize store layout and product placement based on customer behavior.

    Heatmaps and Visualizations:

    • Generate heatmaps to visualize customer flow patterns within the store, highlighting popular areas and bottlenecks.
    • Create bar charts to represent stock levels in inventory, providing a clear view of product availability.
    • Use pie charts to capture and display customer dwell time distribution, helping to identify sections where customers spend the most time.

    VisioAid - Visually Impaired Assistive System

    Description

    VisioAid, designed to enhance the lives of visually impaired. The software incorporates Explainable AI concepts, including robust object detection(vehicles and obstacles), real time voice based explanations, GPS navigation, and drug prescription by processing text and giving medication voice assistance.

    • Market Statistics: The global market for Assistive Technologies targeting the Visually Impaired began at around USD 3.9 billion in 2021, highlighting significant potential for accessibility enhancement. Projected to expand to approximately USD 13.2 billion by 2030, these findings underscore the growing importance of assistive technologies for visually impaired individuals.

    Use Cases:

    Vehicle Detection:

    • In the first module, an integrated camera deploys ensembled output of YOLO v8 and YOLO NAS used for object detection.
    • Additionally, monocular depth estimation is utilized to calculate the distance of the obstacle from the user.
    • These box coordinates, class probabilities, and distance information are passed to GPT Transformer to generate humanized explainable text.
    • Transforming information into speech via gTTS, users receive output as a voice instruction for guidance.

    GPS Navigation

    • The second module leverages GPS coordinates from a receiver using Geolocation API, processed via the Google Maps API.
    • Employing API's like Directions API to fetch best possible route from source to destination, Distance API to get the distance, JavaScript Maps API to fetch realtime maps.
    • The users are directed using voice guided navigation utilizing gTTS

    Drug Prescription

    • Drug labels or tablet strips are captured and processed using OCR and NLP techniques.
    • The captured image is pre-processed by rotating into 8 different possible directions for accurate text recognition.
    • Pytesseract library is utilized to recognize printed text.
    • An explainable AI model clarifies content, such as dosage instructions, conveyed through the speaker.

    Facify - Facial Recognition System for E-commerce Transaction

    Description

    Facify an Enhanced E-commerce security through two-factor authentication with OTP verification and facial recognition model built through a combination of data augmentation, image pre-processing, model pruning and verification with liveness integration capable of detecting frontal and profile faces and determine any spoofs involved with user verification.

    • Use Cases:

    Face Detection and Preprocessing:

    • The code initializes video capture to read frames from a video source (camera).
    • Each frame is processed to detect faces using MTCNN and dlib.
    • Histogram equalization is applied to improve contrast.

    Face Embedding Extraction:

    • The FaceNet model is loaded and pruned by removing small weights, creating a sparser model, and saving it.
    • Detected faces are cropped, resized, prewhitened, and fed into the FaceNet model to extract embeddings.

    Liveness Detection:

    • The anti-spoofing model is used to perform liveness detection on the recognized faces to ensure they are real.

    Result Display and Verification:

    • The embeddings are used to predict the identity of the faces using a pre-trained classifier.
    • The results, including bounding boxes, landmarks, and labels, are displayed.
    • Verification is performed based on the recognized identity and liveness detection.
